# sc2207-lab-assignment
## Pre-requistites:
### Install MySQL

Mac Users:\
**Install Homebrew**: 
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
**Install MySQL**:
```
brew install mysql
brew services start mysql
```

Windows Users:
```
winget install --id=Oracle.MySQLServer
```
